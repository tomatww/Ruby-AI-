# Telegram Ехо-бот

Telegram Ехо-бот — це простий Telegram-бот, створений мовою Ruby з використанням гему `telegram-bot-ruby`.  
Основна функція бота — привітання користувача після команди `/start` та повторення всіх отриманих текстових повідомлень.

## Технології та залежності

- Мова програмування: Ruby 3.x  
- Менеджер пакетів: Bundler  
- Основний гем: `telegram-bot-ruby`  
- Telegram Bot API

Всі залежності описані у файлі `Gemfile`.

## Встановлення та запуск

Для відтворення роботи бота необхідно:

1. Клонувати репозиторій:
git clone https://github.com/tomatww/Ruby-AI-.git
2. Перейти до каталогу проєкту:
cd telegram_project
3. Встановити всі необхідні геми:
bundle install
4. Запустити бота:
ruby bot.rb
або
bundle exec ruby bot.rb

Після цього бот перейде у режим очікування повідомлень та почне працювати з Telegram Bot API.

## Використання

- **/start** — надсилає користувачу вітальне повідомлення.
- **Будь-який текст** — бот повторює текст у форматі:  
  `"Ти написав: <повідомлення>"`.

## Структура проєкту

- `bot.rb` — основний файл з логікою бота  
- `Gemfile` — список залежностей  
- `Gemfile.lock` — зафіксовані версії гемів  
- `README.md` — документація проєкту
